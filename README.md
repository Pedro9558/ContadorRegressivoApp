# Aplicativo Contador Regressivo

Um aplicativo feito para realizar contagens regressivas e alertar o usuário através de um audio especifico contido na aplicação caso o tempo tenha terminado.

## Pré-requisitos

1.Um celular com sistema Android ou emulador que suporte sistemas Android
  > Caso escolha o celular, um cabo USB também será necessário
2.Uma API de Nível 15(Ice Cream Sandwich) ou superior
  > Segue o link com a lista das APIs Android para referencia: https://pt.wikipedia.org/wiki/Hist%C3%B3rico_de_vers%C3%B5es_do_Android#cite_note-6

## Como utilizar
1. Abra o aplicativo "Contador Regressivo"
2. Clique na caixa de seleção para selecionar um som ou nenhum
3. Abaixo da área de digitar um tempo, digite o tempo em horas, minutos e segundos em seus respectivos campos
4. Clique no botão iniciar e espere o contador terminar
